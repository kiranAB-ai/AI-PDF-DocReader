# DocuMind AI 📄🤖

DocuMind AI is an intelligent PDF analysis system that can read documents and answer user questions based on their content.  
It helps users quickly understand and extract information from PDFs using AI.

---

## 🚀 Features
- Upload PDF documents  
- Automatic text extraction  
- AI-based question answering  
- Natural language responses  
- Supports academic and general documents  
- User-friendly interface  

---

## 🛠️ Tech Stack
- Python  
- Flask  
- HTML  
- CSS  
- JavaScript  
- NLP / AI processing  

---

## 📂 Use Cases
- Student study assistance  
- Document understanding  
- Notes extraction  
- Quick information retrieval from PDFs  

---

## ▶️ How to Run
1. Clone the repository  
2. Install requirements  
3. Run Flask app  
4. Upload PDF and ask questions
